Project Goal

The AT Design brand was created for an academic project.
The goal of the project was to create a personal brand that represents all the work I do, including Graphic Design, UX/UI Design and Web Development.

Logo Construction

The final logo was built taking into account 3 principles, creativity (creative work), clarity (product conveys the message clearly) and impulsion (final product boosts the client business) .

Colors

Web Design - Ideas

After completing the brand creation it was time to build the website that would be my portfolio. The first step was to analyze other portfolios and take the best of each one to build a solid product.

Web Design - Prototype

With all the ideas in mind it was time to start building the prototype, where I used Figma. The main idea for this website was to be something simple, aesthetic and that could be visually connected with the brand.

Web Development

With the prototype done it was time to start coding it. I used Visual Studio using mainly HTML, CSS, JavaScript and the framework Bootstrap.
